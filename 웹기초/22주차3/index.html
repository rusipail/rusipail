  • 글자를 치고 엔터를 치면 checkbox와 글자, x 버튼 세개가 등록된다.
  • 이 때 등록된 글자는 select 태그에 option으로 똑같이 등록된다.
  • 등록 순서는 option과 div는 무조건 같아야 한다.
  • option 태그에 따라 등록되는 위치는 연결된 div 태그의 앞쪽에 등록되고, 맨 뒤 일경우 맨 뒤쪽에 등록된다.
  • 등록된 개수는 계속 업데이트 되어야 한다.
  • checkbox 누를 때
    • checkbox를 누를 경우 해당 div의 배경은 빨강, 글자는 흰색이 된다.
    • 또한 div와 연결된 option 태그도 배경 빨강에 글자가 흰색이 된다.
    • 다시 누르면 원래대로 돌아온다.
  • x버튼
  • 해당 div는 삭제되며, div와 연결되어 있던 option 태그도 삭제된다.
  • 절대로 innerHTML에 태그를 문자열로 작성해서는 안되며, createElement와 insertAdjacentElement 만을 사용해야한다.
  • innerHTML에 문자열만 쓰는 것은 가능하다.
